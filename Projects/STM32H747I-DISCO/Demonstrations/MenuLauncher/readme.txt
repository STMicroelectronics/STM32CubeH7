@par Demo Description

The Menu Launcher application is a sliding icons bar solution which provides a menu of supported demos.
It's based on the STM32H747xx and and the TouchGFX APIs. 

This module will allow the user to select a sub-demo from the list of demos displayed:
TouchGFX, STemWin, and EmbedddedWizrad.

For more details please refer to the full demonstration readme.txt file ../readme.txt

@par Keywords

Demonstration, MenuLauncher

@par Hardware and Software environment

  - This demonstration runs on STM32H747xx devices.    
  - This demonstration has been tested with STMicroelectronics STM32H747I-DISCO
    boards.

The binary and Media demonstration files are available from the STM32H747I-DISCO page in the Binary Delivery section.
