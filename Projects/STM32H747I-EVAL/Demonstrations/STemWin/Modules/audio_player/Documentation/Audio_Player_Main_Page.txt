   <b>Introduction</b>
  
   This software together with the suggested combination of STM32 and ST devices can be used,
   for example, to develop Audio Player module or any audio based application.
   The package contains example implementing audio post processing for gain and loudness adjustments.
   
   The Audio Player software runs on the STM32 microcontroller and includes codecs for audio decoders.
