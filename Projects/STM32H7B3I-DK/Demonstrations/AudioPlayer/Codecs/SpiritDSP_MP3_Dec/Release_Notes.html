Overview


The popular MP3 format is part of the formats a professional application should support to ensure the best musical experience.

Therefore, the STM32 Audio - MP3 Decoder Library removes the need for an external MP3 decoding component.
Moreover, unlike many open-source solutions, it is written specifically for STM32, and is therefore highly optimized, leaving room for other application tasks to run concurrently with the audio decoding. For more details refer to: MP3 Decoder User's Guide

This package includes MP3 Decoder libraries, depending on the STM32 device core: it supports STM32 devices with CortexM3, CortexM4 and CortexM7 cores. The CortexM7 version only works on STM32F7xx and STM32H7xx but not on STM32H7Ax.
